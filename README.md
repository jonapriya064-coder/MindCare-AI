## 📖 About the Project

MindCare AI is a full-stack web application that helps users monitor and understand their emotional well-being using **Artificial Intelligence** and **Natural Language Processing**.

Users type how they feel in plain English. The app's ML model — a **TF-IDF + Logistic Regression pipeline** trained on emotion-labeled text — predicts the emotional state (Happy, Sad, Stressed, Angry, or Neutral), displays a confidence score, and delivers personalised wellness recommendations.

Built as a college capstone project, it demonstrates practical integration of machine learning, a relational database, session-based authentication, and a modern responsive UI — all in a single Python/Flask application.

### Prerequisites

- **Python 3.9+** — [Download here](https://www.python.org/downloads/)
- **pip** — comes bundled with Python
- **Git** — [Download here](https://git-scm.com/)

---

### Step 1 — Clone the Repository

```bash
git clone https://github.com/your-username/MindCare-AI.git
cd MindCare-AI
```

---

### Step 2 — Create a Virtual Environment

A virtual environment keeps project dependencies isolated from your global Python installation.

```bash
python -m venv venv
```

**Activate it:**

```bash
# macOS / Linux
source venv/bin/activate

# Windows (Command Prompt)
venv\Scripts\activate

# Windows (PowerShell)
venv\Scripts\Activate.ps1
```

You'll see `(venv)` in your terminal prompt when it's active.

---

### Step 3 — Install Dependencies

```bash
pip install -r requirements.txt
```

---

### Step 4 — Train the ML Model *(first time only)*

This script trains the emotion classifier on labelled text data and saves it as `model/sentiment_model.pkl`. You only need to run this once.

```bash
python train_model.py
```

Expected output:

```
==================================================
  MindCare AI - Model Training
==================================================

📊 Total training samples : 125
📌 Emotion classes        : ['angry', 'happy', 'neutral', 'sad', 'stress']

🔄 Training model ...
✅ Test Accuracy: 92.00%

💾 Model saved to: model/sentiment_model.pkl
✅ Training complete! You can now run: python app.py
```

---

### Step 5 — Run the Application

```bash
python app.py
```

Open your browser and navigate to:

```
http://localhost:5000
```

The SQLite database is created automatically on the first run. No additional database setup is needed.

## 🗄️ Database Schema

The database is auto-created at `database/mental_health.db` on first run.

```sql
-- User accounts
CREATE TABLE users (
    id         INTEGER PRIMARY KEY AUTOINCREMENT,
    name       TEXT    NOT NULL,
    email      TEXT    NOT NULL UNIQUE,
    password   TEXT    NOT NULL,           -- bcrypt hash, never plain text
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Mood tracking entries
CREATE TABLE moods (
    id       INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id  INTEGER NOT NULL,
    emotion  TEXT    NOT NULL,             -- happy | sad | stress | angry | neutral
    journal  TEXT,                         -- optional note
    date     DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Private journal entries
CREATE TABLE journal_entries (
    id       INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id  INTEGER NOT NULL,
    title    TEXT,
    text     TEXT    NOT NULL,
    date     DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Motivational quotes (20 pre-seeded on init)
CREATE TABLE quotes (
    id     INTEGER PRIMARY KEY AUTOINCREMENT,
    quote  TEXT NOT NULL,
    author TEXT
);
```

## 🤖 Machine Learning

### Pipeline

```
User Input → Text Cleaning → TF-IDF Vectorizer → Logistic Regression → Emotion + Confidence
```

1. **Text Cleaning** — lowercase, strip punctuation and extra whitespace
2. **TF-IDF Vectorizer** — `ngram_range=(1,2)`, `max_features=5000`, `sublinear_tf=True`, `stop_words='english'`
3. **Logistic Regression** — `C=1.0`, `max_iter=1000`, `solver='lbfgs'`
4. **Joblib** — serialises the full `Pipeline` object to `model/sentiment_model.pkl`

### Emotion Classes

| Label | Emoji | Colour | Sample Phrases |
|---|---|---|---|
| `happy` | 😊 | `#28a745` | *"I feel joyful and grateful"*, *"I passed my exam!"* |
| `sad` | 😢 | `#6c757d` | *"I feel empty and lonely"*, *"I have no motivation"* |
| `stress` | 😰 | `#fd7e14` | *"My deadlines are killing me"*, *"I can't sleep from anxiety"* |
| `angry` | 😠 | `#dc3545` | *"I am furious and frustrated"*, *"I feel betrayed"* |
| `neutral` | 😐 | `#17a2b8` | *"Today was just a normal day"*, *"Nothing special happened"* |

### Training Data

- **125 labelled samples** (25 per emotion class)
- **80/20 train-test split** with stratification
- Typical test accuracy: **88–96%** depending on random seed
- To improve accuracy: add more samples to `TRAINING_DATA` in `train_model.py` and re-run
